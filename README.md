# Demonstration of multiple Borders on Shapes in JavaFx

[Shapes] can have a single [Stroke].
[Regions] can have a single [Border], which can consist of multiple [BorderStrokes].

This demo seeks to find a way to use [Border] on [Shapes], in order to use multiple Strokes.

## Usage

This demo requires a Java 14+ runtime in the _PATH_.

Run the following to start the demo:

```sh
$ mvn javafx:run
```

[Stroke]: https://openjfx.io/javadoc/14/javafx.graphics/javafx/scene/shape/StrokeType.html

[Border]: https://openjfx.io/javadoc/14/javafx.graphics/javafx/scene/layout/Border.html
